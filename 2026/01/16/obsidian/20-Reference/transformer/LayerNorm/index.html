LayerNorm

发表于2026-01-17|更新于2026-01-17
|浏览量:

并不是BatchNorm简单的交换N和D。

前向传播

在前向传播中，LayerNorm的公式就是BatchNorm交换N和D维度。LayerNorm在每个批次内部进行Normalization，而BatchNorm是跨Batch的Norm。导致mean和std_dev稍有不同。

(1) 均值

μ=1D∑i=1Dxi\mu = \frac{1}{D} \sum_{i=1}^{D} x_i μ=D1​i=1∑D​xi​

(2) 方差

σ2=1D∑i=1D(xi−μ)2\sigma^2 = \frac{1}{D} \sum_{i=1}^{D} (x_i - \mu)^2 σ2=D1​i=1∑D​(xi​−μ)2

(3) 标准差 + eps

σ=σ2+ε\sigma = \sqrt{\sigma^2 + \varepsilon} σ=σ2+ε​

(4) 归一化

x^i=xi−μσ\hat{x}_i = \frac{x_i - \mu}{\sigma} x^i​=σxi​−μ​

(5) 仿射变换（gamma / beta）

yi=γix^i+βiy_i = \gamma_i \hat{x}_i + \beta_i yi​=γi​x^i​+βi​

反向传播

  • 已知 $$\frac{\partial L}{\partial y_i} = d y_i$$
  • 求 $$\frac{\partial L}{\partial x_i},\frac{\partial L}{\partial \gamma_i},\frac{\partial L}{\partial \beta_i}$$

β\betaβ

β\betaβ 在y上进行了广播，所以累计y的梯度就行。

yi=γix^i+βi⇒∂L∂βi=∑dyiy_i = \gamma_i \hat{x}_i + \beta_i \Rightarrow \frac{\partial L}{\partial \beta_i} = \sum dy_iyi​=γi​x^i​+βi​⇒∂βi​∂L​=∑dyi​

γ\gammaγ

简单的乘法

∂L∂γi=∑dyi⋅x^i\frac{\partial L}{\partial \gamma_i} = \sum dy_i \cdot \hat{x}_i∂γi​∂L​=∑dyi​⋅x^i​

dx

传导路径

x → μ → σ → x̂ → y → L

而$$\hat{x}=\frac{x_i - \mu}{\sigma}$$
其中μ\muμ和σ\sigmaσ都依赖于xix_ixi​，所以xix_ixi​一共有三条路径。

  1. 直接贡献

∂x^i∂xi=1σ\frac{\partial \hat{x}_i}{\partial x_i} = \frac{1}{\sigma}∂xi​∂x^i​​=σ1​

  1. μ\muμ的贡献

μ=1D∑kxk⇒∂μ∂xj=1D\mu = \frac{1}{D} \sum_k x_k \Rightarrow \frac{\partial \mu}{\partial x_j} = \frac{1}{D}μ=D1​k∑​xk​⇒∂xj​∂μ​=D1​

  1. σ\sigmaσ的贡献

σ2=1D∑k(xk−μ)2\sigma^2 = \frac{1}{D} \sum_k (x_k - \mu)^2 σ2=D1​k∑​(xk​−μ)2

∂σ2∂xj=2D(xj−μ)\frac{\partial \sigma^2}{\partial x_j} = \frac{2}{D}(x_j - \mu)∂xj​∂σ2​=D2​(xj​−μ)

σ=σ2+ε⇒∂σ∂xj=xj−μDσ\sigma = \sqrt{\sigma^2 + \varepsilon} \Rightarrow \frac{\partial \sigma}{\partial x_j} = \frac{x_j - \mu}{D \sigma}σ=σ2+ε​⇒∂xj​∂σ​=Dσxj​−μ​

  • 最终结果

∂L∂xi=1Dσ(D⋅dx^i−∑jdx^j−x^i∑jdx^jx^j)\frac{\partial L}{\partial x_i}=\frac{1}{D \sigma} \left( D \cdot d\hat{x}_i-\sum_j d\hat{x}_j-\hat{x}_i \sum_j d\hat{x}_j \hat{x}_j \right)∂xi​∂L​=Dσ1​(D⋅dx^i​−j∑​dx^j​−x^i​j∑​dx^j​x^j​)
