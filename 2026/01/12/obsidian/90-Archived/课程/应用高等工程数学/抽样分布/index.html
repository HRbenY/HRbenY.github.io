抽样分布

发表于2026-01-13|更新于2026-01-13
|浏览量:

这三个分布（χ2\chi^2χ2、t、F）被称为数理统计中的**“三大抽样分布”**。

它们都有一个共同的“妈妈”——标准正态分布 N(0,1)N(0,1)N(0,1)。

要理解它们，不要死记公式，而是要记住它们的**“出身”（定义）和“用途”**。

1. 卡方分布 (χ2\chi^2χ2 Distribution) —— 衡量“波动的总量”

  • 出身（定义）：

    假设你有 nnn 个相互独立的标准正态分布变量 Z1,Z2,…,ZnZ_1, Z_2, \dots, Z_nZ1​,Z2​,…,Zn​。

    把它们平方后加起来，这个和就服从自由度为 nnn 的卡方分布：

    X=Z12+Z22+⋯+Zn2∼χ2(n)X = Z_1^2 + Z_2^2 + \dots + Z_n^2 \sim \chi^2(n) X=Z12​+Z22​+⋯+Zn2​∼χ2(n)

  • 直观理解：

    • 因为是“平方和”，所以卡方分布的值永远是非负的（都在 0 右边）。

    • 它衡量的是一组数据的**“波动总能量”**。

    • 图像是右偏（不对称）的，但随着自由度 nnn 增大，它会越来越像正态分布（中心极限定理）。

  • 用途：

    • 跟方差有关：最经典的应用就是样本方差 S2S^2S2 的分布与卡方有关（(n−1)S2σ2∼χ2\frac{(n-1)S^2}{\sigma^2} \sim \chi^2σ2(n−1)S2​∼χ2）。

    • 拟合优度检验：看观测值和理论值差多少（平方差）。

    • 列联表检验：看两个分类变量是否独立。

  • 性质

    • 1.设χ2∼χ2(n)\chi^2 \sim \chi^2(n)χ2∼χ2(n) ,则E(χ2)=n, D(χ2)=2nE(\chi^2) = n,\ D(\chi^2) = 2nE(χ2)=n, D(χ2)=2n
      1. 设χ12∼χ2(n),χ22∼χ2(n)\chi_1^2 \sim \chi^2(n), \chi_2^2 \sim \chi^2(n)χ12​∼χ2(n),χ22​∼χ2(n)，且χ12\chi_1^2χ12​与χ12\chi_1^2χ12​相互独立，则χ12+χ22∼χ2(n1+n2)\chi_1^2 + \chi_2^2 \sim \chi^2(n_1+n_2)χ12​+χ22​∼χ2(n1​+n2​)

2. t 分布 (Student’s t-Distribution) —— “胖尾巴”的正态分布

  • 出身（定义）：

    假设 ZZZ 是标准正态分布，YYY 是卡方分布，且它俩独立。

    那么，ZZZ 除以 Y/n\sqrt{Y/n}Y/n​ 就服从 t 分布：

    T=ZY/n∼t(n)T = \frac{Z}{\sqrt{Y/n}} \sim t(n) T=Y/n​Z​∼t(n)

  • 直观理解：

    • 长得非常像标准正态分布（钟形，对称，中心在0）。

    • 区别：t 分布的尾巴更厚（Fatter Tails），峰更低。

    • 为什么？ 因为正态分布假设我们知道真实的方差 σ2\sigma^2σ2。但现实中我们不知道，只能用样本方差 S2S^2S2 去估计。这个“估计”引入了额外的不确定性，所以导致两端的极端概率变大了（尾巴胖了）。

    • 当样本量 n→∞n \to \inftyn→∞ 时，t 分布就变成了正态分布。

  • 用途：

    • 小样本均值检验：当你不知道总体方差，且样本量较小（比如 n<30n<30n<30）时，用来检验均值（t-test）。

3. F 分布 (F-Distribution) —— 两个方差的“比值”

  • 出身（定义）：

    假设有两个独立的卡方分布 U∼χ2(n1)U \sim \chi^2(n_1)U∼χ2(n1​) 和 V∼χ2(n2)V \sim \chi^2(n_2)V∼χ2(n2​)。

    它们分别除以自己的自由度，然后相除：

    F=U/n1V/n2∼F(n1,n2)F = \frac{U/n_1}{V/n_2} \sim F(n_1, n_2) F=V/n2​U/n1​​∼F(n1​,n2​)

  • 直观理解：

    • 因为卡方代表“方差/能量”，所以 F 分布本质上是**“方差的比值”**。

    • 既然是比值，且卡方非负，所以 F 分布也永远非负。

    • 图像也是右偏的。

  • 用途：

    • 方差分析 (ANOVA)：这是 F 分布最著名的用途。比较多组数据的方差，看它们之间是否有显著差异（本质是在比较“组间差异”和“组内差异”的比值）。

    • 比较两个总体的方差：检验两个总体波动是否一致。

为了方便记忆，你可以把它们看作积木：

  1. 正态分布 (ZZZ)：最基础的积木。

  2. 卡方分布 (χ2\chi^2χ2)：把 nnn 个正态积木平方后粘在一起。

  3. t 分布 (ttt)：上面放一个正态积木，下面放一个“卡方积木的平方根”。

  4. F 分布 (FFF)：上面放一个卡方积木，下面也放一个卡方积木（算比值）。

它们之间还有一个著名的数学关系（考试常考）：

  • 如果你把 t 分布的变量平方，它就变成了 F 分布！

    T∼t(n)  ⟹  T2∼F(1,n)T \sim t(n) \implies T^2 \sim F(1, n) T∼t(n)⟹T2∼F(1,n)

    (理解：分子是 Z2Z^2Z2 也就是 χ2(1)\chi^2(1)χ2(1)，分母是 χ2(n)/n\chi^2(n)/nχ2(n)/n，这不就是 F 分布定义吗？)
