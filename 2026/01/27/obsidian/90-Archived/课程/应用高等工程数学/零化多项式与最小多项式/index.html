零化多项式与最小多项式

发表于2026-01-27|更新于2026-01-27
|浏览量:

这两个概念是线性代数中判断矩阵性质（尤其是能不能对角化）的核心工具。

1. 零化多项式—— “能打中靶子的子弹”

定义：

如果把矩阵 AAA 代入一个多项式 f(x)f(x)f(x)，结果算出来是零矩阵（f(A)=0f(A) = 0f(A)=0），那么这个 f(x)f(x)f(x) 就叫 AAA 的零化多项式。

  • 直观理解：就是能“消灭”矩阵 AAA 的多项式。

  • 特点：零化多项式有无数个。

    • 如果 x2−1x^2 - 1x2−1 能消灭 AAA（即 A2−I=0A^2 - I = 0A2−I=0），那么 (x2−1)(x+3)(x^2 - 1)(x + 3)(x2−1)(x+3) 肯定也能消灭 AAA，因为它包含了一个能让 AAA 变成 0 的因子。

例子：

A2+A=2IA^2 + A = 2IA2+A=2I，移项得 A2+A−2I=0A^2 + A - 2I = 0A2+A−2I=0。

这说明 f(x)=x2+x−2f(x) = x^2 + x - 2f(x)=x2+x−2 就是 AAA 的一个零化多项式。

2. 最小多项式 —— “口径最小的子弹”

定义：

在所有能消灭 AAA 的零化多项式里，次数最低且最高次项系数为 1 的那个多项式，叫最小多项式，通常记为 m(λ)m(\lambda)m(λ) 或 μA(x)\mu_A(x)μA​(x)。

  • 直观理解：它是零化多项式里的“老大”或者“基因”。

  • 两个最关键的性质：

    1. 整除性：任何其他的零化多项式，一定能被最小多项式整除。

      • 也就是：如果 f(A)=0f(A)=0f(A)=0，那么 m(x)m(x)m(x) 一定是 f(x)f(x)f(x) 的因式。

    2. 根的同源性：最小多项式的根，和特征多项式的根完全一样（只是重数可能不同）。

      • 也就是：特征值是 λ1,λ2\lambda_1, \lambda_2λ1​,λ2​，那么最小多项式里一定有 (x−λ1)(x-\lambda_1)(x−λ1​) 和 (x−λ2)(x-\lambda_2)(x−λ2​)。

3. 为什么它们能判断“对角化”？（重中之重）

黄金定理：

一个矩阵 AAA 可对角化   ⟺  \iff⟺ 它的最小多项式没有重根（全是互不相同的单根）。

例子：

  1. 已知 f(x)=x2+x−2=(x−1)(x+2)f(x) = x^2 + x - 2 = (x-1)(x+2)f(x)=x2+x−2=(x−1)(x+2) 是 AAA 的一个零化多项式。

  2. 根据整除性，最小多项式 m(x)m(x)m(x) 必须是 f(x)f(x)f(x) 的因子。

  3. f(x)f(x)f(x) 的因子只有几种可能：

    • x−1x-1x−1 （无重根）

    • x+2x+2x+2 （无重根）

    • (x−1)(x+2)(x-1)(x+2)(x−1)(x+2) （无重根）

  4. 不管 m(x)m(x)m(x) 是哪一个，它都没有重根。

  5. 结论：所以 AAA 一定可以对角化。

4. 举个栗子（彻底搞懂代数重数 vs 几何重数）

为了看清最小多项式和特征多项式的区别，我们要看两个长得很像但性质完全不同的矩阵：

矩阵 A（乖孩子，可对角化）：

A=(2002)A = \begin{pmatrix} 2 & 0 \\ 0 & 2 \end{pmatrix} A=(20​02​)

  • 特征多项式：∣λI−A∣=(λ−2)2|\lambda I - A| = (\lambda - 2)^2∣λI−A∣=(λ−2)2。（代数重数是2）

  • 最小多项式：

    你会发现，只要一次方 A−2IA - 2IA−2I 就已经是零矩阵了。

    所以 m(x)=x−2m(x) = x - 2m(x)=x−2。

  • 判断：m(x)m(x)m(x) 是单根（一次方），所以可对角化。

矩阵 B（坏孩子，Jordan 块，不可对角化）：

B=(2102)B = \begin{pmatrix} 2 & 1 \\ 0 & 2 \end{pmatrix} B=(20​12​)

  • 特征多项式：∣λI−B∣=(λ−2)2|\lambda I - B| = (\lambda - 2)^2∣λI−B∣=(λ−2)2。（跟 A 一模一样！）

  • 最小多项式：

    试一下一次方：B−2I=(0100)≠0B - 2I = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} \neq 0B−2I=(00​10​)=0。

    试一下两次方：(B−2I)2=0(B - 2I)^2 = 0(B−2I)2=0。

    所以 m(x)=(x−2)2m(x) = (x - 2)^2m(x)=(x−2)2。

  • 判断：m(x)m(x)m(x) 有重根（二次方），所以不可对角化。

总结

  • 零化多项式：只要代入 AAA 等于 0 的就是。

  • 最小多项式：次数最低的那个零化多项式。

  • 判据：盯着最小多项式看，只要它的因式里没有 (x−λ)2,(x−λ)3(x-\lambda)^2, (x-\lambda)^3(x−λ)2,(x−λ)3 这种带方幂的项，矩阵就能对角化。
